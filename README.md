## Project Overview
This project demonstrates the use of Microsoft Power BI to analyze sales and marketing performance data from Google Ads, build relationships between tables, and create interactive dashboards for business insights.

## Tools & Skills Used
- Microsoft Power BI
- Power Query (Data Cleaning & Transformation)
- Data Modeling & Relationships
- DAX (Basic Measures)
- Data Visualization

## Dashboard Highlights
- Total Sales by Year
- Average Profit Margin
- Average CTR (Click Through Rate) vs Conversion Rate
- Total Transaction by State 
- Product-based Sales Analysis using Slicer
  
## Dataset
The dataset used for this project is included in the 'PowerBI-Project---Lezzy-Fashion' folder.
